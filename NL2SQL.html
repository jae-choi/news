오늘날 수많은 대형 플랫폼 기업들이 페타바이트 규모의 데이터를 보유하고 있지만, 정작 현업 부서가 이 데이터에 접근하기는 쉽지 않습니다.

이 구현 사례(POC)는 데이터 분석 전문가 없이도 누구나 일상 언어(Natural Language)를 통해 데이터베이스를 조회할 수 있도록, 자기 수정(Self-Correction)이 가능한 NL2SQL 파이프라인과 고성능 캐싱 시스템을 구축한 기술 검증 기록입니다.

1. Problem

비전문가의 데이터 접근 한계와 조직 차원의 분석 병목 현상

데이터 병목 현상(Data Bottleneck)

마케팅, 기획 등 현업 부서에서 데이터 기반 의사결정을 내리려면 항상 데이터 분석가(Data Analyst)의 SQL 작성 지원이 필요했습니다. 이는 분석 리드타임을 지연시키고 조직 전체의 데이터 활용도를 낮추는 치명적인 병목이었습니다.

단순 LLM의 SQL 생성 오류

자연어를 입력받아 즉시 SQL을 짜주는 모델(NL2SQL)을 사용해도, 기업 고유의 데이터 스키마를 잘못 해석하여 실행되지 않는 쿼리(Syntax Error)를 반환하거나 불필요한 LLM 호출로 인해 응답 시간이 길어지는 문제가 있었습니다.

2. Solution

자기 수정(Self-Correction) LLM과 인메모리 캐싱 시스템의 결합

이러한 한계를 극복하기 위해, 자연어를 SQL로 변환하고 구문 오류 발생 시 스스로 오류를 감지하여 쿼리를 재작성(Self-Correction)하는 Gemini 기반의 추론 엔진을 구축했습니다.

동시에 LLM의 API 호출 비용과 지연 시간(Latency)을 해결하기 위해, Compute Engine 인스턴스에 고성능 인메모리 캐시 서버를 배포하여 유사/동일 질의에 대한 초고속 캐시 응답 아키텍처를 결합했습니다.

4. Technical Decisions

응답 속도 개선과 보안 안정성을 위해 채택한 엔지니어링 의사결정

커스텀 RBAC 개발 vs OAuth 2.0 연동

웹 서비스 보안을 위해 별도의 역할 기반 접근 제어(RBAC) 시스템을 자체적으로 구축하면 세밀한 권한 관리가 가능하지만, 개발 기간이 지연되고 사용자 관리 유지보수 비용이 증가합니다.

해결: Google OAuth 2.0 인증 채택 기업 내에 이미 활성화된 Google Workspace의 OAuth 2.0 인증 방식을 프론트엔드에 연동했습니다. 이를 통해 Google API 서버가 권한을 직접 검증하므로, 별도의 인증 시스템을 구축하지 않고도 빠르고 안전하게 엔터프라이즈 인가 문제를 해결했습니다.

단순 코사인 유사도 vs K-Means 결합 캐싱

초기에는 이전 사용자 질문과 현재 질문 간의 '코사인 유사도(Cosine Similarity)'만을 기준으로 캐시 히트(Hit)를 판별했습니다. 그러나 데이터가 누적될수록 전체 캐시를 풀스캔하는 시간이 길어지고 엉뚱한 의도를 유사하다고 판단하는 에러가 발생했습니다.

해결: K-Means 클러스터링 전처리 도입 모든 캐시를 스캔하지 않고, K-means 알고리즘을 사용해 질의 벡터를 의미 단위로 먼저 그룹화(Clustering)했습니다. 사용자의 질의가 들어오면 속한 군집을 먼저 찾고 해당 군집 안에서만 코사인 유사도를 검색하는 2-Step 로직을 구현하여, 검색 효율과 정확도를 동시에 잡아냈습니다.

5. Business Impact

데이터 민주화(Data Democratization) 달성

SQL 문법을 전혀 모르는 마케터와 기획자도 수억 건의 결제 및 로그 데이터에 즉시 접근하여 자연어로 차트를 생성할 수 있게 되었습니다. 데이터 분석 리드타임이 며칠에서 단 몇 분으로 줄어들었습니다.

오류 자가 치유 시스템 증명

  • 자가 수정 (Self-Correction) 로직: 모델이 생성한 쿼리의 Syntax Error를 잡아내고 에이전트가 이를 스스로 수정(Retry)하는 아키텍처를 통해, 런타임 환경에서 쿼리 실행 성공률을 비약적으로 높였습니다.
  • 확장 가능한 캐싱 인프라: 고성능 인메모리 캐시와 K-Means 클러스터링을 결합하여, 향후 전사 솔루션으로 확장될 때 LLM API 비용을 최소화할 수 있는 튼튼한 기반 구조를 확립했습니다.
